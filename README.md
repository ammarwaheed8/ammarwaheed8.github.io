📌 About The Project

This portfolio website was built to highlight my technical skills, projects, and professional journey. It provides a clean and responsive interface where users can explore my work and learn more about the technologies I use.

The goal of this project is to create a strong online presence and make it easier for recruiters, collaborators, and developers to discover my work.
